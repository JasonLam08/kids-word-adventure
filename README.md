# 单词小冒险

儿童英语单词记忆工具原型。它从 10 个起步词开始，根据最近答题正确率、错词数量和掌握度逐步解锁更大的内置词库。

## 功能

- 首次进入必须选择年龄段，并缓存选择。
- 300 个内置单词，按主题、年龄和难度分级。
- 初始仅开放 10 个词，进步后每次解锁 5 个新词。
- 看图选词、听音选图、英中配对、拼写单词、今日复习。
- 答对、答错、完成练习均有 Web Audio 即时音效。
- 使用浏览器 `speechSynthesis` 播放英文发音。
- 家长视图展示开放词数、掌握词数、待复习词、最近正确率和薄弱主题。

## 运行

当前机器没有 `npm`，所以提供了零依赖静态服务器：

```bash
node scripts/serve.mjs
```

如之后安装了 npm，可安装依赖后使用：

```bash
npm install
npm run dev
```

## 构建检查

```bash
node --check src/main.js
node scripts/build.mjs
```
